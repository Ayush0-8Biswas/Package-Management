Package Management in Linux

It is the process of installing, uninstalling, updating packages.

Packages

A 'package' refers to a compressed file archive containing all the files that come with a particular application.

Package Manager

Automate the process of installing, upgrading and removing programs.

PKGBUILD

Example PKGBUILD file

  # Maintainer: Ayush Biswas biswas.ayush03@kgpian.iitkgp.ac.in
pkgname=guessing-game-git
pkgver=1.1
pkgrel=1
pkgdesc="A cool, lightweight and open source guessing game in the terminal."
arch=(any)
url="https://github.com/Ayush0-8Biswas/guessing-game.git"
license=('unknown')
depends=()
makedepends=(git python3)
source=("git+$url")
md5sums=('SKIP')

build() {
	cd guessing-game
}

package() {
	cd guessing-game
	mkdir -p ${pkgdir}/usr/local/bin/
	cp guessing-game.py ${pkgdir}/usr/local/bin/guessing-game
	chmod +x ${pkgdir}/usr/local/bin/guessing-game
}
